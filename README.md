            <h3 align="left"> Features </h3>
            <div align="left">
                - [x] Acumula último número de commit em arquivo txt<br>
                - [x] Pede mensagem de commit ao usuário<br>
                - [x] Lê arquivo txt emoji e acumulador<br>
                - [x] cria um commit concatenando com o emoji sequencial<br>
                - [x] commita, efetua rebase do repositório git<br>
                - [x] executa o push do repositório para github<br>
                - [x] cria novo branch com o nome sequencial e faz checkout<br>
                - [x] escreve no acumulador txt o novo numero de branch para o próximo commit
            </div>
    </div>

<h3> Para rodar o projeto</h3>
<p>
Inicialize seu ambiente virtual. <br>
Em seguida instale os requirements.
</p>

```bash
pip install -r requirements.txt


```
<p>Copie a pasta "gitutils" e arquivo "rebase_branch.bat" para a raiz de seu projeto.<br>
Iniciar o arquivo "branch_atual.txt" dentro da pasta "gitutils" com o número inicial de commits do seu projeto.<br>
ex: escreva "1", se for o primeiro commit.<br> 
</p>

```bash
rebase_branch.bat
```
<p>
Execute o arquivo "rebase_branch.bat"<br>
Digite p<br>
Escreva o comentário commit e pressione ENTER
</p>
